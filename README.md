## Installation from sources

### Requirements

Install required dependencies with the following commands:

```bash
pip install poetry

pip install --upgrade pip

poetry install
```

### Development

#### Pre-commit

In the development mode install pre-commits with the following commands:

```bash
pre-commit
```

or

```bash
pre-commit install
```
